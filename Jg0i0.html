10. 문자열/컬렉션

문자열

String   문자열 자료형
String str = "Hello, World!";

charAt   클래스 객체 문자열에서 해당 인덱스 번호의 문자값
System.out.println( str.charAt( 1 ) );
System.out.println( str.charAt( 7 ) ); 		  e  
  W  

substring   클래스 객체 문자열에서 시작 인덱스나 끝 인덱스를 전달 받아 부분문자열을 반환하는 메서드
System.out.println( str.substring( 7 ) );
System.out.println( str.substring( 7, 11 ) ); 		  World!  
  Worl  

replace   클래스 객체 문자열에서 전달인자의 왼쪽 문자열을 오른쪽 문자열로 바꾼 결과를 반환하는 메서드
System.out.println( str.replace( "Hello", "Hi" ) );
System.out.println( str.replace( "l", "L" ) ); 		  Hi, World!  
  HeLLo, WorLd!  

contains   클래스 객체 문자열에 전달인자 문자열이 포함되어 있는지 여부를 반환하는 메서드
System.out.println( str.contains( "Hello" ) );
System.out.println( str.contains( "Hi" ) ); 		  true  
  false  

indexOf   클래스 객체 문자열에서 전달인자 문자열의 시작 인덱스를 반환하며, 없으면 -1을 반환하는 메서드
System.out.println( str.indexOf( "World" ) );
System.out.println( str.indexOf( "Hi" ) ); 		  7  
  -1  

split   클래스 객체 문자열을 전달인자를 기준으로 분리하고 그 결과를 배열로 전달하는 메서드
String[] token = str.split( ", " );
str Hello, World!   token Hello World!
trim   문자열 앞뒤에 있는 '\n', '\t', ' '와 같은 공백 문자열 제거하는 메서드

Collection

ArrayList   데이터를 쉽게 추가하거나 삭제할 수 있는 가변 길이 리스트로 멀티 쓰레드에서도 사용 가능

Vector   데이터를 쉽게 추가하거나 삭제할 수 있는 가변 길이 리스트로 단일 스레드에서만 사용 가능
Vector list = new Vector( 2 );
list [0]                  
[1]  
add   리스트에 배열 요소를 추가하는 메서드
list.add( "apple" );
list.add( "banana" );
list.add( "carrot" );
list [0] (String객체01)   (String객체01) apple
[1] (String객체02)   (String객체02) banana
[2] (String객체03)   (String객체03) carrot
[3]  
remove   리스트에서 전달받은 인덱스 번호에 있는 객체를 제거하는 메서드
list.remove( 1 );
list [0] (String객체01)   (String객체01) apple
[1] (String객체03)   (String객체03) carrot
[2]  
[3]  
capacity   Vector에서 메모리 공간을 확보한 배열의 크기
System.out.println( list.capacity() );   4  

size   리스트에 저장된 객체의 개수
System.out.println( list.size() );   2  

get   리스트에서 전달받은 인덱스 번호에 있는 객체를 가져오는 메서드
System.out.println( list.get(1) );   carrot  

contains   리스트에 전달받은 객체가 포함되어 있는지 여부를 반환하는 메서드
System.out.println( list.contains( "apple" ) );
System.out.println( list.contains( "banana" ) ); 		  true  
  false  

isEmpty   리스트에 객체가 남아 있는지 여부를 반환하는 메서드
System.out.println( list.isEmpty() );   false  

clear   리스트 비우기
list.clear();
System.out.println( list.isEmpty() ); 		  false  
  true

Collection   데이터를 쉽게 추가하거나 삭제할 수 있는 가변 길이 리스트로 ArrayList나 Vector의 부모 interface

Collections   Collection과 관련된 static 메서드를 정의한 클래스

sort   리스트에 있는 배열 요소들을 정렬
