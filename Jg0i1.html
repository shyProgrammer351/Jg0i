문제

사용자가 입력한 두 이름이 같은 성인지 다른 성인지 판단하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
A양, 이름을 입력하세요: 허난설헌
B군, 이름을 입력하세요: 허균
허난설헌과 허균은 성이 같습니다. 


A양, 이름을 입력하세요: 신사임당
B군, 이름을 입력하세요: 왕건
신사임당과 왕건은 성이 다릅니다. 


A양, 이름을 입력하세요: 김만덕
B군, 이름을 입력하세요: 김유신
김만덕과 김유신은 성이 같습니다.

알고리즘

프로그램 시작 
   이름 2개를 입력 
   이름에서 첫 음절을 비교하기 
      첫 음절이 같으면 성이 같습니다 출력 
      첫 음절이 다르면 성이 다릅니다 출력 
프로그램 종료

프로그램 코드

	public class FamilyName

		// 프로그램 시작

5			String nameB = scan.next();

			// 첫 음절이 같으면 성이 같습니다 출력

7				System.out.print( nameA + "과 " + nameB + "은 성이 같습니다. " );

			// 첫 음절이 다르면 성이 다릅니다 출력 

8			else

9				System.out.print( nameA + "과 " + nameB + "은 성이 다릅니다. " );
