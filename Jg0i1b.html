문제

사용자가 입력한 두 이름이 같은 성인지 다른 성인지 판단하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
A양, 이름을 입력하세요: 허난설헌
B군, 이름을 입력하세요: 허균
허난설헌과 허균은 성이 같습니다. 


A양, 이름을 입력하세요: 신사임당
B군, 이름을 입력하세요: 왕건
신사임당과 왕건은 성이 다릅니다. 


A양, 이름을 입력하세요: 김만덕
B군, 이름을 입력하세요: 김유신
김만덕과 김유신은 성이 같습니다.

코드 빈칸 채우기

public class FamilyName 
{ 
	public static void main( String[] args ) { 
		Scanner scan = new Scanner( System.in ); 
		System.out.print( "A양, 이름을 입력하세요: " ); 
		String nameA = scan.(); 
		System.out.print( "B군, 이름을 입력하세요: " ); 
		String nameB = scan.(); 
	 
		if ( nameA.(0)  nameB.(0) ) 
			System.out.print( nameA + "과 " + nameB + "은 성이 같습니다. " ); 
		else 
			System.out.print( nameA + "과 " + nameB + "은 성이 다릅니다. " ); 
		 
	} 
}
