문제

아래 설명된 의미에 해당하는 용어를 쓰세요 
 초기식, 조건식, 증감식으로 구성되고, 조건식이 참일 때마다 해당 명령문을 실행하는 반복문  

 조건식이 참이면 해당 명령문을 실행하는 선택문  

 반복문에서 오른쪽 배열의 처음부터 끝까지 순서대로 하나씩 왼쪽 변수에 대입하는데 필요한 기호  

 "apple", "banana" 등을 표현하는 문자열 클래스  

 클래스 객체 문자열을 전달인자를 기준으로 분리하고 그 결과를 배열로 전달하는 메서드  

 클래스 객체 문자열에서 전달인자의 왼쪽 문자열을 오른쪽 문자열로 바꾸고 그 결과를 반환하는 메서드  

 클래스 객체 문자열에 전달인자 문자열이 포함되어 있는지 여부를 반환하는 메서드  

 클래스 객체 문자열에서 전달인자 문자열이 시작하는 인덱스 번호를 반환하며, 없으면 -1을 반환하는 메서드  

 클래스 객체 문자열에서 시작 인덱스나 끝 인덱스를 전달 받아 부분문자열을 반환하는 메서드  

 여는 대괄호로 특정 배열 원소를 가리킬 때 사용하는 기호  

 닫는 대괄호로 특정 배열 원소를 가리킬 때 사용하는 기호
